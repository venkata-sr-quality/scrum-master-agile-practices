# Scrum Master & Agile Practices – Senior Practitioner Portfolio

This repository represents how I practice the Scrum Master role
in real Agile teams, focusing on **delivery, transparency,
continuous improvement, and team health**.

This is not a project or Jira dump.
It showcases **Scrum practices, facilitation techniques,
and Agile thinking** derived from real-world experience.

## What This Repository Covers
- Sprint planning facilitation
- Daily standup structure and intent
- Retrospectives that drive real improvement
- Agile metrics and progress visibility
- Coaching teams toward predictable delivery

## Scrum Master Philosophy
A Scrum Master does not manage tasks.
A Scrum Master **enables people, flow, and improvement**.
